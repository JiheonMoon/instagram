복합선택자

스타일의 중첩
- 한개의 선택자를 대상으로 하는 css의 지정이 여러번 일어날 때 어떤일이 일어날까?
- 일반적으로 동일한 선택자를 대상으로 여러개의 css 명령어가 입력되면 모든 명령어가 적용된다
- 적용하려는 스타일이 서로 다를때만 성립함

스타일이 중첩되어 생길 수 있는 문제점
- CSS코드가 중복될경우 항상 마지막에 입력된 명령이 실행됨

캐스케이딩
- CSS Cascading Style Sheet의 약자
- Cascade는 폭포라는 뜻

css에서 cascading은 스타일이 쏟아져 내리는것을 의미

색상을 표현하는 방법
css에서는 배경색, 글자색, 테두리색 등과 같이 색상을 다루는 경우가 자주있음

inherit
영단어로 상속이라는 의미이다
스타일값으로 inherit를 지정하면 부모태그의 색상을 그대로 물려받음
css의 기본값이며 따로 color속성을 지정하지 않는다면 자동으로 지정됨

color name
red green blue 등 색상을 의미하는 단어

RGB 또는 RGBA 코드 Decimal
- RGB 코드는 색상을 표현하는 가장 단순한 코드이다
255 255 255와 같이 3개의 숫자가 묶여있는 순서쌍으로 정의된다
각각 빛의 삼원색인 빨강, 초록, 파란색을 의미한다

- RGBA는 투명도를 포함
- 색상을 의미하는 RGB 뒤에 투명도를 의미하는 alpha가 추가된 형태

16진수 컬러코드(Hex RGB)
- 색상을 16진수로 표현하는 것으로 컴퓨터와 디스플레이의 발달로 사용도가 매우 높아지고 있다.

글자를 표현하는 스타일
- HTML의 본질은 정보 전달이며, 정보전달의 핵심은 텍스트
- 따라서 css에서는 텍스트를 꾸미기위한 온갖 도구들이 준비되어있음

공간을 표현하는 스타일
공간의 단위
CSS에서 주로 사용되는 공간의 단위에 대해 알아보자
1.% 부모 대비 비율

2.auto 브라우저가 자동으로 계산하게 만드는 값
단독으로 사용하는 경우는 흔하지 않음

3.vm, vh
브라우저 창의 해상도를 기준으로 삼는 길이의 단위
100vm는 브라우저의 화면 너비의 100%를 의미하며 30vm는 화면 너비의 30%를 의미
vm, vh를 사용하면 화면의 크기에 상관없이 항상 일정한 비율로 화면에 컨텐츠를 표시할수 있으

크기 지정하기
일반적으로 크기를 지정할때 width와 height속성을 활용

여백지정하기
css의 여백은 외부여백과 내부여백이 있다

1.외부여백 margin
요소 바깥쪽의 여백, 여백에는 다른요소가 침범할수 없음

2.내부여백 padding
요소 안쪽의 여백

영역을 표시하는 스타일
영역의 테두리를 표시하는 border

선의 종류 border
- border 스타일 속성에는 선의 종류를 입력한다
- 선을 없애고 싶다면 스타일 속성 자체를 삭제하거나, 스타일값으로 none을 입력
- 점선, 대쉬, 실선, 겹선 등 다양한 종류의 선을 적용할수 있음

영역의 배경을 표현하는 background
1.배경의 색상을 결정하는 background-color

2.배경에 이미지를 삽입하는 background-image

화면에 표시되는 방식을 표현하는 스타일 visibility, display
표시하거나 숨기는 기능 visibility

화면에 표시될 규격을 정하는 display
block : 한 행을 다 차지한다
inline : 요소 크기만큼의 공간을 차지한다

요소의 배치
display grid 속성
grid 스타일은 css를 사용해 일종의 격자를 그리는 기술이다
그리드를 적용하려면 div 태그를 대상으로 "display:grid" 스타일을 적용하면 된다.

그리드 적용시 주의사항
그리드를 적용할때 반드시 격자의 내용물이 될 구성요소를 한 덩어리의 태그로 표현해야 한다
애매할 경우 <div>로 감싸버리면 된다
